# Advanced Personal Portfolio

This project is my personal portfolio website created for JavaScript Lab 2.

## How to run the project
1. Download or clone the repository
2. Open the project folder
3. Open index.html in a browser (preferably using Live Server in VS Code)

## Technologies used
- HTML
- CSS
- JavaScript

## Features completed
- Contact form with validation (Grade 3)
- Dynamic project gallery with filtering (Grade 4)
- Theme switcher with saved preference (Grade 5)

## Browser support
Tested on:
- Google Chrome
- Microsoft Edge
